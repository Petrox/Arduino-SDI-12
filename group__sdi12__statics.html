uint16_t SDI12::bitTimes ( uint8_t  dt)
staticprivate

#include <src/SDI12.h>

static method for calculating the number of bit-times that have elapsed given an 8-bit counter/timer timestamp.

Parameters
dtThe current value of the 8-bit timer
Returns
uint16_t The number of bit times that have passed at 1200 baud.

Adds a rxWindowWidth fudge factor to the time difference to get the number of ticks, and then multiplies the fudged ticks by the number of bits per tick. Uses the number of bits per tick shifted up by 2^10 and then shifts the result down by the same amount to compensate for the fact that the number of bits per tick is a decimal the timestamp is only an 8-bit integer.

See also
https://github.com/SlashDevin/NeoSWSerial/pull/13#issuecomment-315463522
