SDI-12 for Arduino  2.1.0
An Arduino library for SDI-12 communication without requiring any additional hardware.
g_terminal_window.ino

This is a simple demonstration of the SDI-12 library for Arduino.

It's purpose is to allow a user to interact with an SDI-12 sensor directly, issuing commands through a serial terminal window.

75  Serial.println(sdiResponse); // write the response to the screen
